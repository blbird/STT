# STT training

## Corpus training 
### Add corpus
```sh
node corpus add <corpus name> <corpus file>
```

### list corpus
```sh
node corpus list
```

## Words Training
### Add corpus
```sh
node words add <words file>
```

### list corpus
```sh
node words list
```

## Acoustic training
### Add acoustic file
```sh
node acoustic add <zipped acoustic file>
```

### list corpus
```sh
node acoustic list
```
